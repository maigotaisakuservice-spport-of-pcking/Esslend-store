# Esslend Store (Panda3D Prototype)

これは、ホラーゲーム『Esslend Store』のプロトタイプです。
Panda3DフレームワークとPythonを使用して、仕様書に基づいたコアゲームメカニクスを実装しています。

## 実行方法

### 必要なもの
- Python 3.x
- Panda3D ライブラリ

### インストール
まだPanda3Dをインストールしていない場合は、ターミナルまたはコマンドプロンプトで以下のコマンドを実行してください。
```bash
pip install panda3d
```

### ゲームの起動
リポジトリのルートディレクトリから、以下のコマンドを実行してください。
```bash
python EsslendStore_Panda3D/main.py
```

ゲームが起動し、ウィンドウが表示されます。
コンソール（ターミナル）には、現在のゲームの状態（時刻、異変の有無など）がデバッグ情報として表示されます。
ゲームを終了するには、`Esc`キーを押してください。
